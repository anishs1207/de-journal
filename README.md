# Solana Journal Dapp (DeJournal)

A **simple journaling app** built on the **Solana Devnet** that allows you to create, store, and view your personal journals securely on the blockchain.

## Features

- Create and store journals directly on the **Solana blockchain**.
- Each journal entry is immutable and linked to your wallet.
- Simple and clean **user interface** for easy journaling.
- Fully deployed on **Vercel** for quick access.

## Tech Stack

- **Solana Devnet** – decentralized storage of journal entries
- **Nextjs** – frontend
- **Anchor / Solana SDK** – smart contract interactions
- **Vercel** – deployment

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/anishs1207/de-journal.git
```

2. Run locally

```bash
npm run dev
```
